# 🍽️ Restaurant Website

A clean and responsive static website for a restaurant, built with pure HTML, CSS, and JavaScript — no frameworks, no backend, just fast and elegant frontend.

---

## 📌 Description

**Restaurant Website** is a frontend-only web project designed to present a restaurant online. It includes key sections such as a menu, about page, contacts, and more — all styled with custom CSS and interactive elements powered by vanilla JavaScript.

---

## 🧠 Features

- 🍕 Menu showcase with dishes and categories
- 📖 About the restaurant section
- 📍 Contacts and location info
- 📱 Fully responsive design for mobile and desktop
- ⚡ Fast load - no backend, no dependencies

---

## 🛠 Technologies Used

| Layer    | Technology          |
|----------|---------------------|
| Markup   | HTML5               |
| Styling  | CSS3                |

## 🚀 Quick Start

### Clone the Repository
```bash
git clone https://github.com/babayaga0x/Restaurant_Website.git
cd Restaurant_Website
```

### Open in Browser

Simply open `index.html` in any modern browser:
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Or serve locally with a dev server:
```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

Then visit `http://localhost:8080`.
